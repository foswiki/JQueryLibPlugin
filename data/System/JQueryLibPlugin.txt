---++ Description

Provides [[http://jQuery.com][jQuery]] libraries to Foswiki.
Also includes jQuery UI libraries, themes and some jQuery plugins.

---++ Usage

This plugin does not currently provides any macro.

Once installed and enabled you get extra %TOPIC% settings in %CONFIGURE%.

={Plugins}{JQueryLibPlugin}{Scripts}= let you define the scripts to be added in the HTML head of every pages return by your Foswiki installation. %BR%
={Plugins}{JQueryLibPlugin}{Styles}= let you define the scripts to be added in the HTML head of every pages return by your Foswiki installation.

Scripts and styles will be added to the HTML head in the order provided in the settings.

---++ Settings 

---+++ jQuery UI 1.5

To use jQuery UI 1.5 got %CONFIGURE% and use the following settings:

<verbatim>
{Plugins}{JQueryLibPlugin}{Scripts}=jquery-1.2.6.min.js, jquery-ui-1.5.3.min.js, plugins/farbtastic/farbtastic.js
{Plugins}{JQueryLibPlugin}{Styles}=themes/lightness/ui.all.css, plugins/farbtastic/farbtastic.css
</verbatim>

See Sandbox.JQueryLibPluginDemo for some examples.

---+++ jQuery UI 1.7

To use jQuery UI 1.7 preview got to %CONFIGURE% and use the following settings:

<verbatim>
{Plugins}{JQueryLibPlugin}{Scripts}=jquery-1.3.2.min.js, jquery-ui-1.6rc6.min.js, plugins/farbtastic/farbtastic.js
{Plugins}{JQueryLibPlugin}{Styles}=themes/1.7/smoothness/ui.all.css, plugins/farbtastic/farbtastic.css
</verbatim>

See Sandbox.JQueryLibPluginUI1Dot7Demo for some examples.

---++ Supported jQuery plugins

   * [[http://acko.net/dev/farbtastic][Farbtastic color picker]]: comes with DataForms =color= type support and PatternSkinEasyColorSettings.
